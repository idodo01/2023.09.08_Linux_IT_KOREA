	1. 홈디렉토리 - 로그인한 사용자의 "내문서" 같은 느낌

1) 경로 - /home/<사용자이름>/  (물결표(~) 축약하여 사용)
user@ubuntu:~/Desktop$ cd ~
user@ubuntu:~$ 

+) 참고: root 계정의 홈디렉토리는 /root (/home/root 아님)


	2. 루트 디렉토리

1) 경로 - / (절대경로)
user@ubuntu:~$ cd /
user@ubuntu:/$

2) 개념
최상위 디렉토리 ( 루트 디렉토리보다 상위 디렉토리는 없음)

	3. 경로

1) 절대경로 -user@ubuntu:/$  <- 절대 경로로 시작한다는 것을 의미

: 루트 디렉토리로 명시한 경로 

2) 상대경로
user@ubuntu:~$  : 상대 경로로 시작한다는 것을 의미

	4. cd (change directory): 디렉토리 이동
	5. 상위 디렉토리 (..)
	
사용 4+5 : cd .. : 상위 디렉토리로 이동

+) 현재 작업 디렉토리가 ~ 일때 (~가 사용자 홈디렉토리 위치인데), 
cd.. 하면 작업 디렉토리가 /home으로 변경된다 (그 상위는 /home이니까)

user@ubuntu:~$ cd ..
user@ubuntu:/home$ 

	6. pwd : 현재 작업 디렉토리의 (절대)경로 출력

+) root(/) 안에 /home 안에 user

	11.rmdir : 디렉토리 삭제

	12.rm(remove) : 파일, 디렉토리 삭제

	ex) rm -rf* 
	-f: 대상으로 하는 경로나 패턴에 일치하는 파일이나 디렉토리가 없어도
	-r: 만약 대상 디렉토리가 비어있지 않다면, 내용부터 삭제하고 디렉토리 삭제


	----실습
user@ubuntu:~/school$ mkdir school
user@ubuntu:~/school$ mkdir school/stu1
user@ubuntu:~/school$ mkdir school/stu2
user@ubuntu:~/school$ mkdir school/stu3
user@ubuntu:~/school$ mkdir school/stu4

user@ubuntu:~$ rm school			- 안에 내용이 있어서 삭제불가
rm: cannot remove 'school': Is a directory

user@ubuntu:~$ rm -rf school 		- 안에 내용 제거 후 디렉토리제거

	13. cp(copy) - 파일 혹은 디렉토리 복사

	cp (대상파일) (복사하고자하는 경로)

user@ubuntu:~$ touch file1
user@ubuntu:~$ ls
Desktop    Downloads  linux  Pictures  Templates
Documents  file1      Music  Public    Videos
user@ubuntu:~$ cp file1 file2
user@ubuntu:~$ ls
Desktop    Downloads  file2  Music     Public     Videos
Documents  file1      linux  Pictures  Templates

	14. mv
	mv (대상파일) (복사하고자하는 경로)
(1) 파일 이동

user@ubuntu:~$ cd linux
user@ubuntu:~/linux$ ls
c2  file1  java3  linux1

c2  file1  java3  koreait  linux1
